# Movie Recommendation System

## About
This project is a movie recommendation system built using machine learning.  
It recommends movies based on similarity between their descriptions.

---

## Work Done (50%)
- Collected dataset from Kaggle  
- Loaded and cleaned the dataset  
- Used TF-IDF to convert text data into numerical form  
- Applied cosine similarity to find similar movies  
- Built a function to recommend top movies  
- Saved model files using .pkl  

---

## Technologies Used
- Python  
- Pandas  
- Scikit-learn  
- Google Colab  

---

## How it Works
User enters a movie name and the system recommends similar movies based on content.
